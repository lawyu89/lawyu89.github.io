Gold Grabber Game

I created this game for a challenge problem at DBC when we were first learning how to program with JavaScript. At first I was going to create a simple game with basic functions and objects like the challenge prompt instructed but then I saw an example game created by a student using DOM objects. After seeing how amazing it was I wanted to do something better. It was definitely very challenging in learning DOM manipulation with only a few days of JavaScript but I couldn't stop because of my tenacious nature. I must have spent a whole day to creating my game and researching how to manipulate DOM objects. Anyway, you can try my game here! Hope you enjoy playing it!
