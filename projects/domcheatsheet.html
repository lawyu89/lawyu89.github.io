Changing DOM Elements

From JavaScript you can change elements text, attribute, and their CSS styles.

  • element.innerHTML
  • element.attribute
  • element.style."CSS property"

The .innerHTML method allows JavaScript to alter the text in the html file. The .attribute method changes the element's attribute. And the .style method allows JavaScript to change a CSS property. Handling Events

Javascript can react to certain HTML user events like, clicking, or mousing over and object. These commands call on JavaScript to run its code:

  • onload= : runs whatever function when screen is loaded
  • onunload= : runs whatever function when someone is about to leave page
  • onchange=: runs functions whenever something is changed (like a input or form)
  • onmouseover=: runs functions when mouse is over element
  • onmouseout=: runs functions when mouse is in element and then leaves
  • onfocus=: runs functions when an input field gets focus
  • onresize=: runs functions when browser window is resized
